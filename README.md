# Tara Tech - Website Jasa Pembuatan Website

Website promosi untuk Tara Tech, perusahaan jasa pembuatan website profesional.

## Teknologi yang Digunakan

- Next.js 14
- TypeScript
- Tailwind CSS
- shadcn/ui components
- Lucide React Icons

## Cara Menjalankan

1. Install dependencies:
```bash
npm install
```

2. Jalankan development server:
```bash
npm run dev
```

3. Buka browser dan akses `http://localhost:3000`

## Fitur

- Hero section dengan call-to-action
- Section layanan (Company Profile, E-Commerce, Landing Page)
- Keunggulan Tara Tech
- Form kontak
- Design responsif dengan warna base navy
- Animasi dan efek hover yang smooth

## Build untuk Production

```bash
npm run build
npm start
```
